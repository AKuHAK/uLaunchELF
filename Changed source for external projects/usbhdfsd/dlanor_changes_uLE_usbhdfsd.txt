These are the changes needed to eliminate compiler warnings from the usbhdfsd
project, as previously modified for use with uLE. The source level changes
are identical, but the included files (in addition to this text) differ in
one instance. The file fs_driver.c is changed for the uLE version.

Changes to usbhdfsd.c
---------------------
Added "#include <loadcore.h>" to avoid a compiler warning
NB: This file is identical for original and uLE versions

Changes to usb_driver.c
-----------------------
Added "#include <thbase.h>" to avoid a compiler warning
Modified an XPRINTF call in usb_driver.c:298 due to bad argument list, removing " altSetting=%i" eliminated a compiler warning about too few arguments
NB: This file is identical for original and uLE versions

Changes to fs_driver.c
----------------------
Added "#include <sysmem.h>" to avoid two compiler warnings
Commented out unused variable "index" in 'fs_dopen' function
NB: This file differs between original and uLE versions
Additionally modified the old fixes to fs_dread, for reading timestamp structs, so as to ensure correct handling of the top byte of the 'year' elements.

Changes to scache.c
-------------------
Added "#include <sysmem.h>" to avoid two compiler warnings
Modified an XPRINTF call in scache.c:186 due to bad argument list, inserting "%i" for alignedSector variable eliminated warning about too many arguments
Commented out unused variable 'ret' in function 'scache_allocSector'
Commented out unused variable 'ret' in function 'scache_writeSector'
NB: This file is identical for original and uLE versions

Changes to fat_driver.h
-----------------------
Added the following prototypes to eliminate five warnings:
int fat_mountCheck();
void fat_setFatDirChain(fat_bpb* bpb, fat_dir* fatDir);
int fat_readFile(fat_bpb* bpb, fat_dir* fatDir, unsigned int filePos, unsigned char* buffer, int size);
int fat_getFirstDirentry(char * dirName, fat_dir* fatDir);
int fat_getNextDirentry(fat_dir* fatDir);
NB: This file is identical for original and uLE versions

With all of the above changes the usbhdfsd project compiles without any other
warning than the usual one about import section attributes.
